# This is NOT the Next.js you know

This project uses **Next.js 16**, which has breaking changes from your training data — APIs, conventions, and file structure may differ. Read the relevant guide in `node_modules/next/dist/docs/` before writing any code. Heed deprecation notices.

Known Next 16 gotchas already relied on in this codebase:

- `params` and `searchParams` are **`Promise`s** — you must `await` them in pages and route handlers.
- Route Handlers live in `app/**/route.ts`; they are not cached by default.
- `PageProps<'/route'>` / `RouteContext<'/route'>` are global helpers generated by `next dev`/`next build`/`next typegen`.
